# ARCHITECTURE_DECISIONS

## ADR 001: Next.js Framework
- **Status**: Accepted
- **Context**: Need a full-stack React framework with SSR/SSG capabilities.
- **Decision**: Use Next.js.
- **Consequences**: Fast performance, SEO-friendly, but requires Node.js environment.

## ADR 002: Tailwind CSS
- **Status**: Accepted
- **Context**: Need a scalable styling solution.
- **Decision**: Use Tailwind CSS.
- **Consequences**: Rapid UI development, small bundle sizes.
